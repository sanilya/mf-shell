# mf-shell

Host shell application built with React, TypeScript, Vite, MUI, Tailwind CSS, and module federation.

## Setup

```sh
npm install
```

## Scripts

- `npm run dev` starts the Vite dev server.
- `npm run build` type-checks and builds the app.
- `npm run lint` runs ESLint.
- `npm run preview` serves the production build locally.

## Git Ignore

Generated folders and local files such as `node_modules/`, `dist/`, logs, caches, editor metadata, and environment files are ignored.
